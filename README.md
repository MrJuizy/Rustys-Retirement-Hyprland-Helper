# Rusty-s-Retirement-Hyprland-reserved-Screenspace

### Dependencies
- gamescope
- Rusty's Retirement
- my install script
### Installing

1. Make sure you have the dependencies installed (see above).
2. In this project directory, make the install script executable:

   ```sh
   chmod +x install.sh
   ```

3. Run the install script:

   ```sh
   ./install.sh
   ```
